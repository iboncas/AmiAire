# Commands

## Docker
```docker compose up -d```  
```docker compose down```  

## MongoDB
Connect:  
```docker exec -it tfg-mongodb mongosh -u admin -p admin```  
Other commands:  
```use tfg```  
```show collections```  
Mongo Express (GUI):  
http://localhost:8081

## MinIO
Run MinIO:  
http://localhost:9001 (minioadmin:minioadmin)
